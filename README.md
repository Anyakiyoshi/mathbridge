# MathBridge — 高数全流程交互式带学平台

按照同济大学《高等数学》教材章节编排，将微积分与线性代数/几何思维深度融合。每节包含**直觉引入 → 严格定义 → 代数/几何视角 → 例题 → 经济学案例 → AI 判题练习**的完整学习流程。

## 🚀 快速开始

### 前提条件
- 安装 [Node.js](https://nodejs.org)（≥18 版本）

### 启动

```bash
cd mathbridge
npm install        # 首次运行：安装依赖
npm run dev        # 启动开发服务器
```

浏览器打开 `http://localhost:5173` 即可使用。

> **注意**：按 Ctrl+C 可停止服务器。

### 一键启动（Windows）

双击项目根目录的 **`start.bat`**，自动启动服务器并打开浏览器。

## 📚 十章内容

| 章 | 内容 | 核心可视化 |
|---|------|-----------|
| 1 | **函数与极限** — ε-δ 定义、夹逼准则、连续性 | 极限双带图、间断点分类 |
| 2 | **导数与微分** — 导数=线性变换、链式法则=矩阵乘法、泰勒=基变换 | 割线→切线动画 |
| 3 | **微分中值定理** — 罗尔/拉格朗日/柯西、洛必达法则六型 | 割线与平行切线 |
| 4 | **不定积分** — 原函数族、换元法、分部积分 LIATE 口诀 | 积分曲线族 + 换元对比 |
| 5 | **定积分** — 黎曼和=离散内积、FTC=互逆线性算子 | 黎曼和收敛动画 |
| 6 | **定积分应用** — 旋转体(盘/壳法)、弧长(度量因子) | 圆盘法动画 |
| 7 | **微分方程** — 方向场=向量场的流、矩阵指数=流算子、相图=特征向量几何 | 方向场 + 相图 |
| 8 | **多元微分** — Jacobian=m×n 线性变换、Hessian=二次型、拉格朗日乘数=正交投影 | 等高线 + 梯度下降 |
| 9 | **重积分** — 二重积分=双重线性型、Jacobian 换元、极/柱/球坐标 | 网格细化动画 |
| 10 | **无穷级数** — 判别法全表、幂级数收敛半径、傅里叶=正交基 | 部分和收敛 + 方波合成 |

## 🧠 AI 判题练习

每节末尾有 **📝 练习** 区域，支持 DeepSeek AI 实时出题和评判。

### 配置 API Key

1. 编辑项目根目录的 `.env.local` 文件
2. 填入你的 DeepSeek API Key：
   ```
   VITE_DEEPSEEK_API_KEY=sk-你的key
   ```
3. 重新启动 `npm run dev`

### 使用
- 点击 **🔄 换一题** → AI 根据当前章节知识点实时出新题
- 输入答案 → **📤 提交评判** → AI 打分并给出详细讲解
- 点击 **💬 追问 AI 助教** → 针对题目继续深入讨论

> `.env.local` 已被 `.gitignore` 忽略，不会上传到 GitHub。
